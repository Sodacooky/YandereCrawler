# 简介

C++ yande.re 下载器

输入tags，界定页的范围，然后在yande.re中下载符合输入的tags的图片。

在Linux下你可能需要手动设置CURLOPT_PROXY以使程序通过你的代理。

# 安装与编译

依赖curl、spdlog、nlohmann-json。

如果你使用vcpkg并设置好了环境变量那么你可以直接使用CMake编译，或直接使用VisualStudio。

如果使用CMake，你可能需要自己修改CMake。
