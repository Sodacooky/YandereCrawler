# 简介

C++ yande.re 下载器

输入tags，界定图片ID的范围，然后在yande.re中爬取下载符合输入的tags的图片到程序目录。

# 安装与编译

只使用了curl库，准备一份curl，修改YandereCrawler.h中的#include，然后用你喜欢的方式去编译。

当然，使用vcpkg安装curl更加方便。
